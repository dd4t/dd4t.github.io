Releases

DD4T consists of 2 parts: the templates and the web application support. Web application support is available in two flavors: Java and .NET. The Templates and the .NET application support are always released together, the java application support follows its own trajectory.

The different parts of the solution have independent release cycles. The following table shows the various releases and the dependencies between them.

Templates / .NET Compatible Java versions
2.2 2.1 or higher (not out yet)
2.1 2.0.6 or higher
